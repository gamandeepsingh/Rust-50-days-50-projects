# Rust 50 Days, 50 Projects

Short Rust practice projects, one per day.

## Projects

| Day | Project | Description |
| --- | --- | --- |
| 01 | [Compress](Day_01_compress/) | Gzip-compress a file and print basic file stats. |
| 02 | [Decompress](Day_02_decompress/) | Extract files from a ZIP archive. |
| 03 | [Read CSV](Day_03_read_CSV/) | Parse and read CSV files. |
| 04 | [Read JSON](Day_04_read_JSON/) | Deserialize JSON into Rust structs. |
| 05 | [Write JSON](Day_05_write_JSON/) | Serialize Rust structs into JSON. |
| 06 | [GET Request](Day_06_GET_request/) | Send an HTTP GET request and parse JSON. |
| 07 | [Async Await](Day_07_Async_Await/) | Fetch multiple URLs concurrently with async/await. |
| 08 | [API Calls (CRUD)](Day_08_API_CALLS_CRUD/) | Demonstrate GET/POST/UPDATE/DELETE CLI API calls. |
| 09 | [Axum](Day_09_Axum/) | Build a simple in-memory todo API with Axum. |
| 10 | [Authentication](Day_10_Authentication/) | JWT-based user authentication with register/login endpoints. |
